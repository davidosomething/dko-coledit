Adds a column to the post listing page to search and quick edit custom fields.

== Description ==

* Click the dropdown to search for a custom field by key.
  * You can then edit values (removing focus from a field saves its new value)
  * or Add new values
* Click the plus sign next to the dropdown to create a new key/value pair.

== Installation ==

1. Upload the whole `dko-coledit` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
